# **JZ64** **求1+2+3+...+n**

## 描述

求1+2+3+...+n，要求不能使用乘除法、for、while、if、else、switch、case等关键字及条件判断语句（A?B:C）。

数据范围： 0 < n \le 2000<*n*≤200
进阶： 空间复杂度 O(1)*O*(1) ，时间复杂度 O(n)*O*(*n*)

## 示例1

输入：

```
5
```

返回值：

```
15
```

## 示例2

输入：

```
1
```

返回值：

```
1
```



解题思路：

递归

```c++
class Solution {
public:
    int Sum_Solution(int n) {
        bool x = n>1 && (n+=Sum_Solution(n-1));
        return n;
    }
};
```
